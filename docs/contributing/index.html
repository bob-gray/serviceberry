Contributing

Pull requests welcome. Help make Serviceberry awesome!

If you find a problem or have any questions or ideas, please open an issue.

If you create a Serviceberry plugin, please consider publishing it to npm with a package name that's prefixed with serviceberry- (such as serviceberry-json). In the future I'd like to show a list of plugins on this site.
