Response

class

This object is created internally by Serviceberry and passed as the second argument to handler functions. It is a wrapper object around Node's http.ServerResponse.

Reference

send([options])

Sends a response back to the client. A response can also be implicity sent when the request proceeds and there are no more handlers in the queue.

  • options object [optional]

    Sets response options before finishing request and sending response.

    • status number or object [optional]

    • headers object [optional]

    • body any [optional]

    • encoding string [optional]

    • finish function [optional]

      Listener for http.ServerResponse event finish
